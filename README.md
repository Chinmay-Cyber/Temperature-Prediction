# Temperature-Prediction
This project builds a machine learning model to predict temperature using historical climate data. The dataset is loaded using Pandas and cleaned by removing unnecessary columns and handling missing values. The data is reshaped using the melt function to convert year columns into a single "Year" column with corresponding temperature values. The year values are then cleaned and categorical features such as Area and Months are encoded using Label Encoding so they can be used by the model. The dataset is split into training and testing sets, and a Random Forest Regressor is trained to learn patterns in temperature based on area, month, and year. The model's performance is evaluated using Mean Absolute Error (MAE), and finally the trained model is used to predict temperature for a given location, month, and year.
